# Waaiio Product Roadmap

## Future Features (Backlog)

### Polls v2
- Weighted voting (VIP/loyalty customers get 2x vote weight)
- Auto-action on results (e.g., winner updates menu automatically)
- Reaction-based voting (reply with emoji instead of buttons)
- Recurring polls (e.g., "Send every Friday at 9am")
- Share results as image card for social media

### WhatsApp Flows (Native Forms)
- Migrate booking flow to native WhatsApp Flow (single form instead of multi-step)
- Migrate ordering flow to native WhatsApp Flow
- Create Flow JSON definitions in Meta Business Suite
- Requires Meta approval per flow

### Voice Transcription Enhancements
- Language detection from audio (Whisper auto-detects)
- Transcription confidence scoring
- Store transcripts in chat history for dashboard view

### WhatsApp Catalog API
- Sync products to native WhatsApp catalog for rich browsing
- Keep custom ordering flow for payment/variants/addons
- Catalog for browsing only, bot takes over for checkout

### Multi-language v2
- More languages (Arabic, Portuguese, Hindi, Swahili)
- Per-business language preferences
- Translated dashboard UI

### Max Order Quantity
- Per-product max order quantity setting
- Enforce in ordering flow

### Public FAQ Page
- `/faq` marketing page (currently 404)
- Pull from help center content

### Email Rate Limiting
- Per-business monthly email limits by tier
- Track in email_usage table

### Storage Quotas
- Per-business storage limits (Free: 100MB, Growth: 1GB, Premium: 10GB)
- Check before file uploads

### AI Enhancements
- Ace: help businesses with marketing copy
- Ace: suggest optimal pricing based on category/location
- AI-powered customer insights summary
